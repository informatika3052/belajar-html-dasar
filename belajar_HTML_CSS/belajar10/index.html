BELAJAR PEMOGRAMAN WEB SEMESTER 4

Pemrograman Web, lebih sering disingkat PW atau PWeb, adalah mata kuliah yang mempelajari tentang bagaimana cara membuat web. Untuk tahu cara membuat web, kita harus mengetahui bahasa pemrograman web dan software-software pembuat web. Pertama-tama kita akan diajarkan bahasa-bahasa yang terlibat dalam pemrograman web. Misalnya HTML, XML, XHTML, CSS, Javascript, Ajax, Php, JQuery, dan lain-lain.
